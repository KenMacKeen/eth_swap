# EthSwap — Instant ERC-20 Token Exchange

A decentralized exchange (DEX) DApp that lets users instantly buy and sell a custom ERC-20 token (DAPP) for ETH at a fixed rate, with no order book or counterparty required.

## How It Works

EthSwap holds a reserve of DAPP tokens in its smart contract. Users connect via MetaMask and swap ETH for tokens (or tokens back for ETH) at a hardcoded rate of **1 ETH = 100 DAPP**.

- **Buy:** Send ETH → receive DAPP tokens
- **Sell:** Approve + send DAPP tokens → receive ETH back

All swaps are on-chain and atomic. The `EthSwap` contract emits `TokensPurchased` and `TokensSold` events on each trade.

## Prerequisites

- [Node.js](https://nodejs.org/)
- [Truffle](https://trufflesuite.com/) — `npm install -g truffle`
- [Ganache](https://trufflesuite.com/ganache/) — local Ethereum blockchain
- [MetaMask](https://metamask.io/) browser extension

## Getting Started

**1. Install dependencies**

```bash
npm install
```

**2. Start a local blockchain**

Open Ganache and create a new workspace, or run `ganache-cli`.

**3. Compile and deploy the contracts**

```bash
truffle migrate --reset
```

This deploys `Token` and `EthSwap`, then transfers all 1,000,000 DAPP tokens to the exchange contract.

**4. Run the test suite**

```bash
truffle test
```

**5. Start the frontend**

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) and connect MetaMask to your local Ganache network.

## Smart Contracts

### `Token.sol`

A minimal ERC-20 implementation.

| Property | Value |
|---|---|
| Name | DApp Token |
| Symbol | DAPP |
| Total Supply | 1,000,000 |
| Decimals | 18 |

Implements `transfer`, `approve`, `transferFrom`, and the corresponding events.

### `EthSwap.sol`

The exchange contract.

| Property | Value |
|---|---|
| Name | EthSwap Instant Exchange |
| Rate | 100 DAPP per 1 ETH |

**`buyTokens()`** — `payable`. Calculates `tokenAmount = msg.value * rate`, checks the contract has sufficient tokens, and transfers them to the caller.

**`sellTokens(uint _amount)`** — Caller must first call `Token.approve(ethSwapAddress, _amount)`. The contract calls `transferFrom` to collect the tokens, then sends back the equivalent ETH (`_amount / rate`).
